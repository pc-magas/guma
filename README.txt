How to excecute GUMA.
a)Just go to the path where run.sh and guma.jar is located together.
In UNIX like systems (eg Linux FreeBSD) you can do it witd teminal command cd.

b)If not (located together) bring them together under the same folder.

c)Then run the script run.sh for linux or run.bat for windows.
